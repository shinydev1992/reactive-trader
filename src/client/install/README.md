
# OpenFin installers

## Deprecation Notice

The installers in this directory are redundant. There is no longer the need to generate
the installers and commit them to Git. Instead, installers can be generated on the fly
by supplying OpenFin's installer generator at https://install.openfin.co/ with our own
config generator located at https://web-demo.adaptivecluster.com/openfin/app.json

> **WARNING: Don't move or rename the installers in this directory. These are still linked to
from other sites and moving/renaming would break the links.**
